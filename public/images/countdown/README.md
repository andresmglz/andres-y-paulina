# Countdown Assets

Assets opcionales para llevar la seccion de cuenta regresiva a un look mas escenico.

Archivos esperados:

- `countdown-left-floral.png`
- `countdown-right-floral.png`
- `countdown-left-candle.png`
- `countdown-right-candle.png`

Uso recomendado:

- Flores o bouquets con tonos dusty blue, ivory y terracotta suave.
- Velas o candelabros con fondo transparente.
- PNG o WebP con transparencia.

Fuentes sugeridas para descargar y adaptar:

- Ornamental divider, Public Domain:
  https://freesvg.org/index.php/decorative-divider-with-floral-theme
- Floral ornamental divider, Pixabay:
  https://pixabay.com/vectors/floral-ornamental-divider-ornament-4148560/
- Botanical wedding PNGs:
  https://pikwizard.com/s/png/botanical-wedding/
- Blue bouquet PNGs:
  https://pikwizard.com/s/png/blue-bouquet/
- Ornamental divider SVG, CC0:
  https://freepngimg.com/svg/106607-ornamental-divider-design-13

Nota:

- Verifica licencia y atribucion antes de publicar si descargas desde terceros.
- Si un asset no viene transparente, se puede recortar o regenerar despues.
